Raven
=====

Raven is a Requests wrapper to log into the University of Cambridge Raven portal.


Installation
------------
	
.. code-block:: bash

	$ pip install raven


Documentation
-------------

Detailed documentation available at `erinhengel.com <http://www.erinhengel.com/software/raven/>`_. 


Quickstart
----------

.. code-block:: python

	>>> import raven
	
	...
